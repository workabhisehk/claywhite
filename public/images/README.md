# Image Assets

Please place your images in this directory with the following filenames:

## Required Images

1. **Logo**: `wh-advisory-logo.svg` or `wh-advisory-logo.png`
   - Recommended format: SVG (scalable) or PNG with transparent background
   - Recommended dimensions: At least 200px width, or vector format

2. **Clay White Photo**: `clay-white.jpg` or `clay-white.png`
   - Recommended format: JPG or PNG
   - Recommended dimensions: At least 800px width, aspect ratio 4:5 or 3:4
   - High-quality professional headshot

## Supported Formats

- `.svg` (preferred for logos)
- `.png` (with transparent background for logos)
- `.jpg` or `.jpeg` (for photos)
- `.webp` (optimized format, can be used as alternative)

## Image Optimization

After adding images, consider:
- Using Next.js Image Optimization (automatic with `<Image>` component)
- Compressing large files for web (recommended max 500KB per image)
- Using WebP format for better compression
